機能説明図

よりそいの核となる4つの主要機能

🎤 音声認識システム

医療特化の音声認識エンジンで、診察中の会話をリアルタイムに文字起こし。専門用語も高精度で認識します。

  • ✓ 医療専門用語対応（精度78%→92%向上）
  • ✓ リアルタイム文字表示
  • ✓ 雑音除去・音質最適化
  • ✓ 多言語対応（日本語・英語・中国語など）
  • ✓ 話者識別（医師・患者・家族）
🤖 AI要約エンジン

大規模言語モデルを活用し、診察内容を「要点3行」に自動要約。医療者・患者それぞれに最適化された内容を生成。

  • ✓ 医療者向けSOAP形式自動整形
  • ✓ 患者向けやさしい日本語変換
  • ✓ 重要度に応じた情報階層化
  • ✓ 次回確認事項の自動抽出
  • ✓ 専門用語の自動解説付与
🔐 同意管理システム

録音・共有の同意を三層で管理。透明性を保ちながら、患者のプライバシーを最大限保護します。

  • ✓ 三層同意（録音・院内・家族共有）
  • ✓ リアルタイム同意状態表示
  • ✓ ワンクリック同意変更・取り消し
  • ✓ 監査ログ自動生成
  • ✓ アクセス履歴の双方向可視化
👨‍👩‍👧‍👦 家族共有機能

セキュアなURLで家族・介護者と情報共有。期限・範囲・アクセス権を細かく制御できます。

  • ✓ 期限付きセキュアURL発行
  • ✓ 共有範囲の個別設定
  • ✓ 既読状況のリアルタイム確認
  • ✓ 役割分担機能（誰が何をいつまで）
  • ✓ 緊急時即時失効機能
